RESPONSIVE WEB DESIGN

Done with HTML, JS, mySQL, CSS

Responsive Web App

Created by Trina Chua

Design Concept:

On this homepage, the theme is Singapore Travel Tour Attraction. To capture the attention of the user, I included a video in the background showing a tropical island, enhancing the website's visual appeal with subtle motion.

Additionally, I highlighted three popular travel destinations as the user scrolls down. Filters and animations on images make these sections more engaging and interactive.

For smaller screens, navigation links are tucked into a hamburger menu to avoid clutter. The video background is also optimized for mobile by switching to an image for improved performance. The grid layout adapts to a single-column view for a cleaner, more accessible mobile experience.
